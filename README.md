# 🔐 Password Generator

A simple and secure Password Generator project developed using Python.  
This application generates strong and random passwords to improve account security.

---

## 📌 Features

✅ Generate random passwords  
✅ User-defined password length  
✅ Supports:
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

✅ Beginner-friendly project  
✅ Fast and lightweight  

---

## 🛠️ Technologies Used

- Python 3
- random module
- string module
